# 旅游信息推荐管理系统 java1058

## 项目概述

旅游信息推荐管理系统是一个基于Spring Boot和Vue的前后端分离的旅游信息管理系统，旨在提供景区信息、旅游线路、美食信息等的展示和管理。

## 技术栈

- 前端：Vue
- 后端：Java, Spring Boot
- 数据库：MyBatis, MySQL

## 功能模块

### 前台功能

- 展示旅游新闻
- 展示景区信息
- 展示美食信息
- 展示旅游线路
- 在线留言

### 后台管理员功能

- 账号管理
- 地区管理
- 景点信息管理
- 地方美食管理
- 旅游线路管理
- 订单信息管理
- 新闻管理
- 系统管理

### 普通用户功能

- 查看旅游线路
- 查看订单信息

## 系统角色

- 前台游客
- 后台管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8+, MySQL 5.7+

（具体部署步骤省略，实际操作时需根据实际情况补充）
